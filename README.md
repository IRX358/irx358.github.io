## Installation

Follow these steps to get your site up and running:

1. **Fork or Clone the Repository**
   - Fork the repository to your own GitHub account or clone it locally:
     ```bash
     git clone https://github.com/IRX358/irx358.github.io.git
     ```
2. **Modify Content**
   - Open the project in your favorite code editor.
   - Edit `index.html`, Markdown files, or content sections as needed.
   - Update images, styles, and scripts to match your branding.

3. **Preview Locally**
   - Open the `index.html` file directly in your browser to preview changes.
   - For advanced development, use a local web server (e.g., with Python):
     ```bash
     python -m http.server
     ```

4. **Commit and Push Changes**
   - Save your changes and push them to your GitHub repository:
     ```bash
     git add .
     git commit -m "Update website content"
     git push
     ```

5. **Enable GitHub Pages**
   - Navigate to the repository settings on GitHub.
   - Scroll to the "Pages" section and configure the source branch (typically `main` or `master`).
   - After a few minutes, your site will be live at `https://<your-username>.github.io/`.

6. **(Optional) Set Up Custom Domain**
   - Add your custom domain to the GitHub Pages settings.
   - Update your DNS records to point to GitHub Pages.
