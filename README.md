---

## 🏗 Build Setup

Clone the repository:

```bash
git clone https://github.com/IRX358/irx358.github.io.git
```
Install dependencies:
```
npm install
```
Run locally for development:
```
npm run dev
```
Build for production:
```
npm run build
```
The production build will appear in the dist/ folder.
